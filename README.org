#Title: SABDA
_____________________________________________
#What is  SABDA? 
### ~ Synced Nepali Lyrics Collection (Community collected) 

Sabda is the collection of timestamped synced lyrics, mostly in [LRC Format](https://en.wikipedia.org/wiki/LRC_%28file_format%29). The *LRC* format is text-based and similar to subtitle files.

_____________________________________________

##Sample *.LRC* 
### ~ Default format

```
[ti:Title Here]
[ar:Artist Name]
[al:Album Name]
[la:NE]

[00:00.00] Start!
[01:10.00] la la la..
[02:20.00] ho ho ho..
[03:30.00] hu hu hu..
[04.00.23] The End!
```
